## 命名

### 变量命名
全部驼峰
let 小写
const 大写
禁止var

### 函数命名
语义化/驼峰
isXXX ()
setXXX ()
getXXX ()
addXXX ()

### CSS命名
语义化/'-'
vue 文件中css名尽量以组件名 命名
层次清楚
如： .container
      .contatiner-item
        .container-item-title
